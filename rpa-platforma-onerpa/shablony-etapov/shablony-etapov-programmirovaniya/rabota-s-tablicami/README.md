# Работа с таблицами

![](<../../../../.gitbook/assets/Работа с таблицами - общее.png>)

Раздел "Работа с таблицами" необходим для оперирования данными, представленными в виде таблиц, в контексте робота. Например, если нужно данные из нескольких таблиц Excel соединить в одну непосредственно в ходе выполнения робота. Это потребует соединения достаточно большого количества этапов. Но это возможно. Раздел включает в себя:

* [Добавить колонку таблицы](dobavit-kolonku-tablicy.md)
* [Добавить строку таблицы](dobavit-stroku-tablicy.md)
* [Получить значение из таблицы](poluchit-znachenie-iz-tablicy.md)
* [Получить значения строки таблицы](poluchit-znacheniya-stroki-tablicy.md)
* [Получить количество строк таблицы](poluchit-kolichestvo-strok-tablicy.md)
* [Создать описание колонок таблицы](sozdat-opisanie-kolonok-tablicy.md)
* [Создать пустую таблицу](sozdat-pustuyu-tablicu.md)
